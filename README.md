# homebrew-tap

Homebrew tap for [rxliuli](https://github.com/rxliuli)'s apps.

## Usage

```bash
brew install --cask rxliuli/tap/window-resizer
```

## Casks

| Cask | Description |
| --- | --- |
| [window-resizer](https://github.com/rxliuli/window-resizer) | Menu bar utility to resize the active window to preset dimensions |
